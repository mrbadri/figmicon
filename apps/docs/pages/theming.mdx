# Theming

Figmicon supports theming to help you customize the look and feel of your application.

## Theme Configuration

You can customize the theme by providing a theme object:

```jsx
import { ThemeProvider, createTheme } from '@iconsync/core';

const theme = createTheme({
  colors: {
    primary: '#0070f3',
    secondary: '#ff4081',
    success: '#00c853',
    error: '#f44336',
  },
  fonts: {
    body: 'Inter, system-ui, sans-serif',
    heading: 'Inter, system-ui, sans-serif',
  },
  fontSizes: {
    xs: '0.75rem',
    sm: '0.875rem',
    md: '1rem',
    lg: '1.125rem',
    xl: '1.25rem',
    '2xl': '1.5rem',
    '3xl': '1.875rem',
    '4xl': '2.25rem',
  },
});

export default function App({ Component, pageProps }) {
  return (
    <ThemeProvider theme={theme}>
      <Component {...pageProps} />
    </ThemeProvider>
  );
}
```

## Dark Mode

Figmicon supports dark mode out of the box:

```jsx
import { ThemeProvider, createTheme } from '@iconsync/core';

const lightTheme = createTheme({
  // Light theme configuration
});

const darkTheme = createTheme({
  // Dark theme configuration
});

export default function App({ Component, pageProps }) {
  const [isDarkMode, setIsDarkMode] = useState(false);
  const theme = isDarkMode ? darkTheme : lightTheme;

  return (
    <ThemeProvider theme={theme}>
      <Component {...pageProps} />
    </ThemeProvider>
  );
}
```
